Principle

We take the protection of your personal data very seriously and treat personal data confidentially and in accordance with the statutory data protection regulations and this privacy policy.

Collecting Data

We collect and process personal data only insofar as this is necessary for the establishment and content of or change in the legal relationship (inventory data). We do not collect personal data on the use of our app (usage data).

All data in our apps are stored primarily on the respective mobile device. Only when using the synchronization function will the data be transferred via the internet to Google Drive servers and from there to the connected devices. Google Drive uses Google's privacy policy.

Submitting Data

We do not transfer personal data to third parties.

App Permissions

  • Photos / Media / Files
  • Needed for viewing photos and playing music.

  • Storage
  • The PirateBox needs this permission to write and read configuration and store files.

  • Wireless Connection Information
  • Required by the application to display the current WLAN status.

  • Other-Vibration
  • A vibration can be triggered via API. Furthermore, a vibration is used when pressing the start / stop button.

  • Other-Change System Settings
  • For developers, it is possible to change the device's API system settings.

  • Other-Write Web Bookmarks and History
  • The API allows you to write and read Web bookmarks.

  • Other-Show Other Network Connections
  • For the hotspot functionality it is necessary to monitor the network connections.

  • Other-Change Other Network Connectivity
  • Required to start the network hotspot.

  • Other-Connect and Disconnect Wi-Fi
  • Required for the provision of the hotspot.
